## [2026-03-05] - feature/#237-layout-setup-KSY

### Added & Changed (추가 및 변경 사항)

**1. 주요 증권사(토스증권 등) UI 레퍼런스 분석**
  * **사용자 친화적 탭 구조 파악:** 토스증권의 직관적인 종목 상세 인터페이스를 분석하여 차트, 호가, 종목정보, 거래현황으로 이어지는 사용자 흐름을 도출했습니다.
  * **정보 우선순위 재정립:** 모바일 앱의 높은 사용성을 웹 환경에 이식하기 위해 사용자가 가장 빈번하게 확인하는 데이터(현재가 및 차트)를 상단에 배치하는 구조를 채택했습니다.

**2. 전체 페이지 레이아웃 및 와이어프레임 설계**
  * **AI 예측 패널 전용 영역 확보:** 기존 증권 서비스의 일반적인 '우측 주문 영역' 대신, AssetMind의 핵심 기능인 'AI 예측 패널'을 우측 사이드바 형태로 배치하여 서비스의 정체성을 강화했습니다.
  * **반응형 그리드 시스템 적용:** 데스크톱 및 태블릿 환경에서 메인 차트 영역과 AI 패널이 시각적으로 충돌하지 않도록 2컬럼 레이아웃을 기반으로 가독성을 확보했습니다.

**3. 종목 상세 정보 내비게이션 최적화**
  * **상태 전환 인터페이스 설계:** 탭 메뉴(차트/호가/정보/거래) 간 전환 시 사용자가 현재 위치를 명확히 인지할 수 있도록 액티브 상태의 시각적 강조 효과를 적용했습니다.
  * **데이터 확장성 고려:** 향후 추가될 수 있는 다양한 기술적 지표나 예측 데이터가 레이아웃의 일관성을 해치지 않도록 유연한 컴포넌트 구조를 설계했습니다.
