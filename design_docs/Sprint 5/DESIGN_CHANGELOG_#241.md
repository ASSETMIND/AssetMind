## [2026-03-11] - feature/#241-ai-prediction-panel-KSY

### Added & Changed (추가 및 변경 사항)

**1. 고밀도 AI 가격 예측 우측 패널 레이아웃 설계**
* **공간 최적화**: 320px의 좁은 우측 패널 제약 내에서 미니 스파크라인 차트, 예측 목표가, 분석 근거 등을 계층화하여 배치했습니다.
* **정보 밀도 조절**: 핵심 지표인 '예측 목표가'와 '방향성 확률'을 상단에 배치하여 사용자가 스크롤 없이도 핵심 인사이트를 즉시 파악할 수 있도록 설계했습니다.

**2. 예측 가격 범위 및 방향성 확률 시각화**
* **Linear Gauge 활용**: 상승과 하락 확률을 직관적인 바 형태의 게이지로 디자인하였으며, AssetMind의 컬러 시스템을 적용하여 시각적 일관성을 확보했습니다.
* **미니 스파크라인**: 예측 가격의 흐름을 보여주는 차트 영역에 목표가를 레이블링하여 가격 변동성을 직관적으로 시각화했습니다. (현재는 대략적인 시안만 잡아둔 상태로 추후 구체화 예정입니다.)

**3. 다중 분석 근거 위젯 컴포넌트 디자인**
* **상태별 시각적 구분**: 기술적·시장 심리·수급 분석 항목을 긍정(녹색), 중립(회색), 경고(노란색) 상태로 구분하여 리스트 형태로 디자인했습니다.
* **탭 기반 상세 정보 분리**: 분석 카테고리를 탭 구조로 설계하여 정보 과부하를 방지하고 사용자가 관심 있는 데이터만 선택적으로 탐색할 수 있도록 구현했습니다.

**4. 예외 케이스(Skeleton 및 Error) UI 대응**
* **스켈레톤 UI 설계**: 데이터 로딩 시 발생할 수 있는 레이아웃 흔들림을 방지하기 위해 각 컴포넌트의 형태를 반영한 회색 블록 처리의 로딩 상태 시안을 제작했습니다.
* **에러 상태 가이드**: 네트워크 오류 등으로 데이터 로드 실패 시, 직관적인 에러 아이콘과 함께 사용자의 행동을 유도하는 '다시 시도' 버튼을 포함한 예외 UI를 구성했습니다.
