# 실시간 데이터 피드백 UI/UX 디자인 명세

본 문서는 웹 해상도 기준 테이블 레이아웃 내 실시간 주가 데이터 갱신 시 적용되는 컬러 및 애니메이션(Ticker, Flicker) 정책을 정의합니다. (Issue #184)

## 1. 컬러 정책 (Color Tokens)
다크 모드 기준으로 웹 접근성 명도비(AA 등급, 4.5:1 이상)를 충족하는 컬러를 사용합니다.

### 텍스트 컬러 (Text)
* 상승 (Rise): `#EA580C` (Orange-600)
* 하락 (Fall): `#256AF4` (Blue-500)

### 배경 하이라이트 컬러 (Background)
데이터 갱신 시 테이블 행(Row) 배경에 순간적으로 적용되는 컬러입니다.
* 상승 갱신 (Update-Rise): `#EA580C` (투명도 10%)
* 하락 갱신 (Update-Fall): `#256AF4` (투명도 10%)

---

## 2. 애니메이션 명세 (Animation Specs)
실시간 데이터 갱신 시 시각적 피드백을 제공하기 위해 다음 두 가지 애니메이션을 동시에 적용합니다.

### A. Ticker (숫자 롤링 효과)
가격 및 등락률 데이터가 갱신될 때, 기존 숫자가 프레임 위로 밀려나며 새 숫자가 아래에서 올라오는 슬롯머신 형태의 효과입니다.
* 대상: 현재가, 등락률 등 실시간 변동 텍스트
* 동작: 아래에서 위로 슬라이드 (Slide Up)
* 지속 시간: 300ms
* 속도 곡선: Ease-out

### B. Flicker (행 배경 깜빡임 효과)
체결 데이터가 갱신되는 순간 사용자의 시선을 유도하기 위해 해당 종목의 행(Row) 전체 배경색이 변했다가 원래 상태로 부드럽게 돌아가는 효과입니다.
* 대상: 갱신이 발생한 테이블 행(`tbl-row`) 전체 영역
* 동작: 갱신 즉시 배경색(10% 불투명도) 적용 후, 일정 시간 대기하다가 원래 배경(투명) 상태로 페이드아웃(Fade-out)
* 대기 시간: 200ms
* 지속 시간: 150ms
* 속도 곡선: Ease-out
