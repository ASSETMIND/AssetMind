# AssetMind Design System

AssetMind 주식 정보 서비스의 디자인 시스템 Storybook입니다.
컴포넌트 탐색, 상태별 시나리오 검증, Mock 데이터 교체까지 이 페이지 하나로 시작하세요.

---

## 빠른 시작

```bash
cd apps/web/Design_UI
npm run storybook
# → http://localhost:6006
```

## 역할별 사용 가이드

### Developer

**Props 탐색**
각 컴포넌트 Story의 **Docs 탭**을 열면 자동 생성된 Props 테이블(타입·기본값·설명)을 확인할 수 있습니다.

**상태 재현**
우측 **Controls 패널**에서 `pageState` / `status` / `viewport` / `isMarketClosed`를 실시간으로 전환하세요.

**Mock 데이터 교체**
각 `*.stories.tsx` 상단의 상수를 수정합니다.

```ts
// StockDataPage.stories.tsx
const MOCK_ROWS: StockRow[] = [
  { id: "1", rank: 1, name: "삼성전자", price: 75000,
    changeRate: 2.35, tradeAmount: 980000, buyRatio: 62, isFavorite: true },
  // ...
];

// StockDetailPage.stories.tsx
const MOCK_STOCK = { ticker: "005930", name: "삼성전자",
  price: 75000, changeFromYesterday: 1720, changeRate: 2.35 };
```

**반응형 확인**
Storybook 상단 툴바의 뷰포트 아이콘 → `Mobile1 (393px)` / `Tablet (768px)` 전환

---

### Design

- **컴포넌트 모든 상태 확인**: Controls의 `status` = `skeleton` / `error` / `empty` 전환
- **컬러 토큰 명도 대비**: `StockDataPage → QA — Color Contrast Check` Story
- **레이아웃 비교**: `StockDetailPage → Desktop / Tablet / Mobile` 각 Story 나란히 확인

**isMarketClosed 전파 확인**
`StockDetailPage → Desktop / Market Closed` Story에서 `isMarketClosed=true` 시 아래 3개 패널이 동시 전환되는지 확인하세요.
- TradingViewWrapper → 우측 상단 휴장 시간 뱃지
- OrderbookTable → 호가 데이터 숨김 + 뱃지
- AIPredictionPanel → 매수하기 버튼 → 휴장 시간 뱃지

**탭 전환 검증**
`StockDetailPage → Desktop / Tab Switching Demo` 또는 `Mobile / Tab Switching Demo` Story에서 직접 탭을 클릭해 전환 동작을 확인하세요.
