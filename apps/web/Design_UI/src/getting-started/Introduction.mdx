# AssetMind Design System

AssetMind 주식 정보 서비스의 디자인 시스템 Storybook입니다.
컴포넌트 탐색, 상태별 시나리오 검증, Mock 데이터 교체까지 이 페이지 하나로 시작하세요.

## 자주 묻는 질문

**Q. 차트가 빈 화면으로 나와요.**
`TradingViewWrapper`의 `data` prop이 비어 있을 때 발생합니다. `generateChartData(180)`으로 Mock 데이터를 생성해 전달하세요.

**Q. 모바일 Story가 너무 길게 보여요.**
데코레이터에 `height: 852px` + `overflow: hidden`이 적용되어 있어야 합니다. `StockDetailPage` 모바일 Story는 이미 적용되어 있으며, 신규 Story 작성 시 동일하게 설정하세요.

**Q. InvestorTradePanel 모바일에서 매수/매도가 세로로 쌓이지 않아요.**
`panelWidth={345}`로 전달되어야 합니다. `StockDetailPage`에서는 viewport 분기로 자동 처리됩니다.

**Q. StockInfoPanel 좌측 탭이 모바일에서 보이지 않아요.**
`panelWidth <= 345`일 때 좌측 탭이 숨김 처리됩니다. 모바일에서는 탭 없이 콘텐츠만 표시되는 것이 정상 동작입니다.

**Q. CategoryModal이 열리지 않아요.**
`BusinessItem`의 `modalProps`가 정의되어 있어야 합니다. `modalProps`가 없는 항목은 클릭해도 모달이 열리지 않습니다.

**Q. isMarketClosed=true인데 AI 패널에 뱃지가 안 보여요.**
`StockDetailPage.tsx`의 `resolvedAIStatus` 처리가 적용된 버전(2026-04-09 이후)을 사용하고 있는지 확인하세요.

## 신규 컴포넌트 Story 추가 방법

1. `src/components/common/stock/{ComponentName}/` 폴더 생성
2. `ComponentName.tsx` + `ComponentName.stories.tsx` 작성
3. title을 `"Components/Stock/{ComponentName}"` 으로 설정
4. `tags: ['autodocs']` 추가 → Docs 탭 자동 생성
5. **Default / Skeleton / Error / Playground** 4종 Story 기본 구성
