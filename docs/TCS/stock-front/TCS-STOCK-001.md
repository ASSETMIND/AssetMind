## 1. 개요 (Overview)

본 문서는 AssetMind 주식 도메인의 실시간 랭킹 기능에 대한 UI 컴포넌트, 비즈니스 로직(Hooks), API 모킹 및 STOMP 기반 웹소켓 통합 동작을 검증하기 위한 단위/통합 테스트 명세이다.
사용자가 화면에 진입하여 웹소켓에 연결되고, 실시간으로 주식 데이터를 구독/수신하며, 이를 UI 요구사항(포맷팅, 게이지 비율 등)에 맞춰 정확하게 렌더링하고 해제하는 전체 라이프사이클을 확인한다.

### 1.1. 테스트 환경

- **Framework:** Jest, `@testing-library/react` (React Hooks Testing Library)
- **Mocking Strategy:**
  - `MSW (Mock Service Worker)`를 활용한 REST API 엔드포인트 격리
  - `@stomp/stompjs` 라이브러리 모킹을 통한 실제 웹소켓 네트워크 요청 차단
  - 컴포넌트 단위 테스트 시 하위 자식 컴포넌트 및 비즈니스 훅 모킹 분리
- **Key Verification:**
  - **Integration Flow:** STOMP 연결 -> 구독 -> 메시지 수신 -> UI 데이터 가공(콤마, 억원 단위, 비율 제한) 연계 확인
  - **Hooks Logic:** 등락률에 따른 동적 스타일링(깜빡임 애니메이션), 비정상 데이터 방어 로직
  - **UI Render & Event:** 탭 필터 클릭에 따른 상태 변경, 동적 width 할당, 조건부 렌더링
