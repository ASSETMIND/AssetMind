01

화면 구조 분석 (Layout Anatomy)

종목 세부 정보 페이지는 수직 고정 헤더 + 탭 내비게이션 + 좌우 2단 본문으로 구성됩니다. 좌측은 콘텐츠 탭 영역, 우측은 주문하기 패널(310px 고정)이 항상 노출되는 레이아웃입니다. 탭을 전환해도 우측 패널은 교체되지 않습니다.

01
글로벌 네비게이션
홈·피드·주식 골라보기·내 계좌 탭 + 검색 + 로그인 버튼 — 전 페이지 공통 고정 헤더
02
종목 헤더
종목 로고 + 이름 + 코드 + 현재가 + 전일비 + 52주 최고/최저 — 탭 전환 시에도 고정 노출
03
메인 탭 바
차트·호가 / 종목정보 / 뉴스·공시 / 거래현황 / 커뮤니티 — 언더라인 스타일 탭
04
콘텐츠 탭 영역
탭에 따라 교체되는 메인 콘텐츠 — 차트·호가·종목정보·거래현황 각각 고유 레이아웃 보유
05
우측 고정 패널
주문하기 패널 (310px 고정) — 탭 전환에 무관하게 항상 노출. → Assetmind에서 AI 예측 패널로 대체
06
하단 시세 Ticker
코스피·코스닥·나스닥 등 주요 지수 실시간 스크롤 바 — 페이지 하단 고정
레이아웃 핵심 관찰

우측 고정 패널은 토스증권의 수익화 핵심 영역(주문 전환)이지만, Assetmind의 AI 예측 서비스 관점에서는 동일한 위치와 고정 너비(310px)를 활용하여 AI 가격 예측 패널을 배치하는 것이 자연스러운 대안입니다. 사용자 시선이 이미 이 위치에 주목하도록 학습되어 있습니다.

좌우 2단 레이아웃의 너비 비율 및 구조는 다음과 같습니다.

콘텐츠 영역 (Fluid)
차트·호가·종목정보·거래현황 탭 콘텐츠
너비: 나머지 전체 (calc(100% - 310px))
우측 고정 패널 (Fixed 310px)
주문하기 / → AI 예측 패널
탭 전환 시에도 상시 노출

2-1. 차트·호가 탭

Candlestick Chart

  • 이동평균선 오버레이 (MA 5·20·60·120)

  • 크로스헤어 + OHLCV 툴팁 (hover 인터랙션)
  • TradingView 위젯 내장 추정 (로고 워터마크 확인)

03

UX 패턴 및 디자인 원칙

📐

고정 우측 패널 전략

주문 패널은 탭 전환에 무관하게 항상 노출됩니다. 사용자의 매매 전환을 위한 의도적 고정 설계로, AI 예측 패널에도 동일하게 적용 가능한 패턴입니다.

🔁

탭 내 서브 탭 계층

차트·호가 탭 내부에서 다시 차트/호가 서브 탭이 존재합니다. 정보 계층을 2단으로 분리하여 콘텐츠 밀도를 높이면서도 화면을 단순하게 유지합니다.

⏱

실시간성의 다층 표현

체결 Ticker 애니메이션, 호가 실시간 갱신, 투자자 동향 타임스탬프 등 다양한 방식으로 실시간성을 시각적으로 전달합니다.

🇰🇷

한국 금융 색상 관행

파랑=상승·매수, 빨강=하락·매도 원칙이 차트·호가·테이블 전반에 일관되게 적용됩니다. 글로벌 관행과 반대이므로 프로젝트 전체에서 통일 필요합니다.

🗂

종목정보의 SPA 패턴

좌측 메뉴 클릭 시 우측 콘텐츠만 교체되는 SPA 방식으로 전체 페이지 리로드 없이 섹션 이동이 가능합니다. 페이지 전환 이질감 최소화 전략입니다.

📊

데이터 시각화 다양성

캔들·라인 차트·도넛 차트·수평 바·수직 바·테이블 등 다양한 시각화 방식을 한 페이지에 혼합합니다. 정보 유형에 맞는 최적 시각화를 선택합니다.

핵심 인사이트

토스증권 종목 세부 페이지의 핵심은 탭 구조로 복잡한 정보를 계층화하면서도 우측 패널은 항상 고정하여 핵심 행동(주문)에 대한 접근성을 유지하는 것입니다. Assetmind에서는 이 고정 패널을 AI 예측 패널로 전환함으로써, 사용자의 주의가 이미 집중된 위치에 AI 인사이트를 자연스럽게 노출하는 전략이 가능합니다.

04

설계 적용 제안

레퍼런스 분석을 바탕으로 Assetmind 종목 세부 정보 페이지에 적용할 핵심 설계 방향을 도출했습니다.

Keep (유지)

그대로 채택할 패턴

  • 차트·호가 / 종목정보 / 거래현황 3탭 구조
  • 언더라인 스타일 메인 탭 + 서브 탭 2단 계층
  • 기간 선택 Pill 탭 (1분·일·주·월·년)
  • 호가창 배경 바 잔량 시각화
  • 종목정보 좌측 섹션 네비게이션 + 우측 콘텐츠
  • 거래원 매매 상위 수평 바 차트
  • 투자자별 3선 라인 차트 + 순매수 테이블
  • 하단 실시간 지수 Ticker 스트립
Replace & Add (교체·추가)

Assetmind 고유 변경 사항

  • 교체 우측 주문 패널 → AI 가격 예측 패널
  • 추가 AI 예측 목표가 + 상승/하락 확률 게이지
  • 추가 AI 분석 근거 카드 (기술적·감성·수급 탭)
  • 추가 관련 종목 추천 카드 컴포넌트
  • 추가 매수/매도 비율 Linear Gauge
  • 추가 Skeleton / 에러 상태 UI
