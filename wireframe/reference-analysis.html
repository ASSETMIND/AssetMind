01

화면 구조 분석 (Layout Anatomy)

토스증권 실시간 차트 페이지는 수직으로 명확하게 구분된 5개의 레이어로 구성되어 있습니다. 각 레이어는 독립적인 기능을 담당하며 위에서 아래로 자연스럽게 정보를 전달합니다.

01
글로벌 네비게이션
국내장/해외 데이마켓 탭 — 최상위 시장 분류. 현재 선택 상태를 색상(주황 점)으로 명확히 표시
02
광고 배너 영역
여백을 명시적으로 "광고 공간"으로 표시 — 수익화를 위한 의도적 레이아웃 설계
03
서브 탭 네비게이션
실시간 차트 / 지금 뜨는 카테고리 / 국내 투자자 동향 — 콘텐츠 뷰를 전환하는 탭
04
필터 컨트롤 바
전체/국내/해외 + 정렬 기준(거래대금, 거래량 등) + 기간(실시간/1일/1주일…) — 3단 필터 레이어
05
데이터 테이블
순위 / 종목명·로고 / 현재가 / 등락률 / 거래대금 / 토스 거래 비율 바 — 핵심 콘텐츠 영역
02

컴포넌트 상세 분석

Navigation Tabs

탭 네비게이션 패턴

  • 밑줄 + 색상 변화로 현재 탭 표시 (언더라인 스타일)
  • 글로벌 탭은 활성 상태를 색상 점(•)으로 추가 강조
  • 탭 클릭 시 콘텐츠 영역 전체가 교체되는 방식
  • 모바일 대응: 가로 스크롤 가능한 탭 바 구조
Filter Controls

필터 칩(Chip) 패턴

전체 국내 해외
  • 선택 상태: 진한 배경 + 흰 텍스트 (solid fill)
  • 미선택: 테두리 없는 텍스트만 표시 (ghost)
  • 정렬 기준과 기간 필터가 같은 행에 혼재
  • 현재 활성 정렬 기준이 강조 표시됨
Data Table Row

테이블 행 구조 분석

컬럼 내용 비고 중요도
순위 숫자 + 즐겨찾기(하트) 아이콘 즐겨찾기는 호버 또는 기본 표시 높음

03

UX 패턴 및 디자인 원칙

🎯

정보 밀도 최적화

행 높이를 최소화하면서도 종목 로고, 플래그, 비율 바 등 풍부한 정보를 한 행에 담음. 스캔 가능성을 최우선으로 설계.

🔗

컨텍스트 보존 필터링

필터를 변경해도 스크롤 위치와 테이블 컨텍스트가 유지됨. 사용자가 탐색 흐름을 잃지 않도록 설계.

⚡

실시간성 강조

"오늘 11:44 기준"과 같은 타임스탬프로 데이터 신선도를 명시. 실시간 뱃지와 시간 표기로 신뢰성 확보.

🇰🇷

한국 금융 관행 준수

파랑=상승, 빨강=하락의 한국 증권 관행을 정확히 따름. 글로벌 관행(빨강=상승)과 반대이므로 명확한 적용 필요.

📱

모바일 퍼스트 구조

세로 중심 레이아웃, 대형 터치 타깃, 수평 스크롤 가능한 필터 바. 앱과 웹이 동일한 구조를 공유.

🏷

플랫폼 고유 지표

거래 비율 바는 토스 내부 데이터로만 구성되는 독자적 지표. 경쟁사와의 차별화 포인트로 작동.

핵심 인사이트

토스증권 UI의 핵심 전략은 데이터 밀도와 가독성의 균형입니다. 복잡한 금융 데이터를 다크 테마 + 색상 코딩 + 아이콘 계층으로 분해하여 비전문 투자자도 빠르게 스캔할 수 있도록 합니다. 이 패턴은 우리 페이지 설계에 그대로 적용할 수 있는 검증된 방향성입니다.

04

설계 적용 제안

레퍼런스 분석을 바탕으로 우리 페이지에 적용할 핵심 설계 방향을 도출했습니다.

Keep (유지)

그대로 채택할 패턴

  • 3단 필터 레이어 (분류 > 정렬 > 기간)
  • 순위 + 즐겨찾기 복합 셀
  • 테이블 행 기반 데이터 표현
  • 거래 비율 바 (매수/매도 시각화)
  • 타임스탬프로 데이터 신선도 명시
Improve (개선)

개선할 포인트

  • 필터 칩의 선택 상태를 더 명확하게 시각화
  • 테이블 헤더 고정 (Sticky Header) 추가
  • 로딩 스켈레톤 UI 추가
  • 접근성(ARIA) 레이블 보강
