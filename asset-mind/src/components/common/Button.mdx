# Button Component

버튼은 사용자가 클릭하여 작업을 실행하게 하는 가장 기본적인 인터랙션 요소입니다.
AssetMind의 디자인 시스템 가이드라인을 준수하여 제작되었으며, `variant`와 `size`를 통해 다양한 스타일을 지원합니다.

<br/>

## 설치 (Installation)

```tsx
import { Button } from '@/components/common/Button';
```

<br/>

## 1. 메인 로그인 & 가입 (Main Actions)

로그인 및 회원가입 페이지에서 사용되는 메인 버튼입니다.

### 로그인 버튼

기본 상태, 로딩, 비활성화 상태를 지원합니다.

<div style={{ marginTop: '16px', marginBottom: '16px', padding: '24px', backgroundColor: '#0F1014', borderRadius: '8px', border: '1px solid #2D2E33' }}>
  <div style={{ width: '451px' }}>
    <button 
      style={{ 
        width: '100%',
        height: '52px',
        backgroundColor: '#131316',
        color: 'white',
        fontSize: '16px',
        fontWeight: 500,
        borderRadius: '9px',
        border: 'none',
        cursor: 'pointer'
      }}
    >
      로그인
    </button>
  </div>
</div>

**사용 예시:**
```tsx
<Button variant="primary" size="lg" fullWidth className="h-[52px]">
  로그인
</Button>

// 로딩 상태
<Button variant="primary" size="lg" fullWidth className="h-[52px]">
  <Loader2 className="w-5 h-5 animate-spin" />
</Button>

// 비활성화 상태
<Button 
  variant="primary" 
  size="lg" 
  fullWidth 
  disabled 
  className="h-[52px] bg-[#1C1D21] text-[#5F616B]"
>
  로그인
</Button>
```

### 가입하기 버튼

회원가입 페이지의 메인 액션 버튼입니다.

<div style={{ marginTop: '16px', marginBottom: '16px', padding: '24px', backgroundColor: '#0F1014', borderRadius: '8px', border: '1px solid #2D2E33' }}>
  <div style={{ width: '451px' }}>
    <button 
      style={{ 
        width: '100%',
        height: '52px',
        backgroundColor: '#131316',
        color: 'white',
        fontSize: '16px',
        fontWeight: 500,
        borderRadius: '9px',
        border: 'none',
        cursor: 'pointer'
      }}
    >
      가입하기
    </button>
  </div>
</div>

## 2. 소셜 로그인 (Social Icons)

구글, 카카오 등 소셜 로그인을 위한 원형 아이콘 버튼입니다.

<div style={{ marginTop: '16px', marginBottom: '16px', padding: '24px', backgroundColor: '#0F1014', borderRadius: '8px', border: '1px solid #2D2E33' }}>
  <div style={{ display: 'flex', gap: '16px' }}>
    <div 
      style={{ 
        width: '56px',
        height: '56px',
        backgroundColor: 'white',
        borderRadius: '50%',
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        cursor: 'pointer'
      }}
    >
      G
    </div>
    <div 
      style={{ 
        width: '56px',
        height: '56px',
        backgroundColor: '#FEE500',
        borderRadius: '50%',
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        cursor: 'pointer'
      }}
    >
      K
    </div>
  </div>
</div>

**사용 예시:**
```tsx
<Button variant="google" size="icon" className="w-[56px] h-[56px]">
  <GoogleIcon className="w-[40px] h-[40px]" />
</Button>

<Button variant="kakao" size="icon" className="w-[56px] h-[56px]">
  <KakaoIcon className="w-[40px] h-[40px]" />
</Button>
```

## 3. 폼 액션 (Form Actions)

회원가입 폼 내부에서 사용되는 작은 버튼들입니다.

### 중복 확인

아이디 중복 확인용 버튼입니다.

<div style={{ marginTop: '16px', marginBottom: '16px', padding: '24px', backgroundColor: '#0F1014', borderRadius: '8px', border: '1px solid #2D2E33' }}>
  <button 
    style={{ 
      width: '100px',
      height: '38px',
      backgroundColor: '#6D4AE6',
      color: 'white',
      fontSize: '14px',
      fontWeight: 400,
      borderRadius: '9px',
      border: 'none',
      cursor: 'pointer'
    }}
  >
    중복 확인
  </button>
</div>

### 인증번호 전송

휴대폰 인증번호를 전송하는 버튼입니다.

<div style={{ marginTop: '16px', marginBottom: '16px', padding: '24px', backgroundColor: '#0F1014', borderRadius: '8px', border: '1px solid #2D2E33' }}>
  <button 
    style={{ 
      width: '121px',
      height: '38px',
      backgroundColor: '#6D4AE6',
      color: 'white',
      fontSize: '14px',
      fontWeight: 400,
      borderRadius: '9px',
      border: 'none',
      cursor: 'pointer'
    }}
  >
    인증번호 전송
  </button>
</div>

### 인증 확인

인증 완료 시 비활성화되는 버튼입니다.

<div style={{ marginTop: '16px', marginBottom: '16px', padding: '24px', backgroundColor: '#0F1014', borderRadius: '8px', border: '1px solid #2D2E33' }}>
  <button 
    style={{ 
      width: '100px',
      height: '38px',
      backgroundColor: '#6D4AE6',
      color: 'white',
      fontSize: '14px',
      fontWeight: 400,
      borderRadius: '9px',
      border: 'none',
      cursor: 'pointer'
    }}
  >
    인증 확인
  </button>
</div>

**사용 예시:**
```tsx
// 중복 확인
<Button className="bg-[#6D4AE6] text-white w-[100px] h-[38px]">
  중복 확인
</Button>

// 인증번호 전송
<Button className="bg-[#6D4AE6] text-white w-[121px] h-[38px]">
  인증번호 전송
</Button>

// 인증 확인 (완료 상태)
<Button className="bg-[#1C1D21] text-[#5F616B] w-[100px] h-[38px]" disabled>
  인증 완료
</Button>
```

## Accessibility

- **키보드 네비게이션**: `Tab` 키로 포커스 이동, `Enter` 또는 `Space`로 클릭
- **스크린 리더**: 버튼 텍스트가 자동으로 읽힙니다
- **비활성화 상태**: `disabled` prop 사용 시 접근 불가
