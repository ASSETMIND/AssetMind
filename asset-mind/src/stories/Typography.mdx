# Typography

**Font Family:** Pretendard, Pretendard Variable, sans-serif

## Headings & Body

tailwind.config.js에 정의된 텍스트 스타일입니다.

<br/>

## 사용 예시

```tsx
// Tailwind 클래스 사용
<h1 className="text-l1">큰 제목</h1>
<p className="text-b1">본문 텍스트입니다.</p>
<span className="text-l4">작은 보조 텍스트</span>
```
